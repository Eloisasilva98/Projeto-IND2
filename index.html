Decodificador

O objetivo desse projeto é criar uma aplicação capaz de codififar e descodificar mensagens utlizando tanto Cifra de Cesar quanto Base64.

A Cifra de César é um dos métodos conhecidos mais antigos de criptografia. É muito simples - é apenas substituição no alfabeto. A transformação é denominada ROTN, onde N é o valor de substituição e ROT é de "ROTAÇÃO" pois trata-se de uma substituição cíclica.

Já a segunda,Base64, é um método de representação baseado em 64 caracteres imprimíveis para representar dados binários.
